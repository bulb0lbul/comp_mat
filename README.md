# Расчёт орбиты небесного тела 1233 Kobresia (1931 TG2). 
В файле **forward.cpp** описано решение прямой задачи построения орбиты и сравнение с координатами обсерватории. Результат работы кода в терминале приведен в **forward_terminal_result.txt**.

В файле **backward.cpp** реализована обратная задача. Результат работы кода в терминале приведен в **backward_terminal_result.txt**. Отклонения для каждого наблюдения приведены в **residuals_ra_dec.csv**.  Нарисованные графики см. папку **plots**.
