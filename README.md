# Расчёт орбиты небесного тела 1233 Kobresia (1931 TG2). 
1. Данные из обсерваторий в ITRF (геоцентрическая система координат). Преобразования происходят при помощи prepare_obs.py
2. Данные эфемид переходят из эклиптической барицентрической системы в экваториальную барицентрическую систему (ICRS). Преобразуются данные при помощи convert_to_horizon.csv

# Как работает прямая задача:
1. Инициализация и загрузка данных.
   
2. Интегрирование траектории астероида.
  Используется метод Дормана-Принса 5-го порядка с адаптивным шагом.
  Правые части уравнений движения (ускорения) вычисляются в функции acc_barycentric, где учитывается гравитация Солнца и всех загруженных планет.
  Интегрирование выполняется в барицентрической ICRS.
Результаты (времена и состояния) записываются в StepRecorder.

3. Построение сплайн-траектории.
По записям интегрирования строится кубический сплайн SplineTraj, позволяющий получать положение астероида в любой момент времени внутри интервала интегрирования.

4. Обработка наблюдений.
  Для каждого наблюдения из файла prepared_observations.csv выполняется:
  a) Преобразование времени.
    Исходное время в UTC преобразуется в TDB.
  b) Определение положения обсерватории в ICRS.
    ITRF → GCRS → ICRS.
  c) Решение уравнения светового времени.
  d) Вычисление топоцентрического направления.
  e) Релятивистские поправки к направлению.
  f) Перевод в сферические координаты.
